Chain - это последовательность логических контроллеров, передающих управление 
по очереди, с дополнительными условиями запуска для каждого этапа.

Каждый этапы цепочки имеет свой автоматически формируемый GUID, для того 
чтобы обеспечить возможность связи логики этапов между собой с учетом 
возможности их перемещения по списку.

Chain - все цепочка этапов с логикой
ChainStage - этап цепочки, содержит условие и какую-то логику
Condition - условие для запуска этапа. Если условие не выполнено, обязанность 
повторного контроля лежит на коде условия.
ChainsController - центральная шина для доступа к цепочкам

Цепочка активируется, автоматически запуская первый из своих этапов.
